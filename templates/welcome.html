Gestion et création

Consulter
  • les généalogies existantes (list)
  • l'historique des erreurs et statistiques (errors)
Sauvegarder
  • un fichier source GEDCOM (gwb2ged)
  • un fichier source GeneWeb (gwu)
Créer une généalogie
  • à partir d'un fichier source GEDCOM (ged2gwb)
  • à partir d'un fichier source GeneWeb (gwc)
  • à partir d'un fichier vide (gwc)


Options avancées
  • Nettoyage (cleanup)
  • Renommage (rename)
  • Suppression (delete)
  • Fusion de généalogies (merge)
  • Sauvegarde et restauration (gwu)
  • Calcul de la consanguinité (consang)
  • Mise à jour des liens fiche/chronique familiale (update_nldb)
  • Calcule les composantes connexes (connex)
  • Calcule la différence entre deux bases (gwdiff)
  • Corrige les problèmes d'une base (gwfixbase)
Configurer
  • Les paramètres d'une base de données (.gwf)
  • Les paramètres du service gwd (gwd)
  • Les fichiers cache de noms et places pour l'auto-completion (cache_files)
